Amazon Web Scraper Project

Python BeautifulSoup Automation

Introduction

Learn how to use web scraping to monitor product prices on Amazon with Python. This project targets a data analyst-themed T-shirt. Using BeautifulSoup and Requests libraries, analysts can automate price tracking and receive email notifications when prices drop below a set threshold.

Project Steps

1. Importing Necessary Libraries

Import essential Python libraries including BeautifulSoup, Requests, time, datetime, and smtplib.

2. Connecting to Amazon Product Page

Define the URL for the T-shirt and set headers to mimic a web browser request. Fetch HTML content with Requests and parse it with BeautifulSoup.

3. Extracting Product Information

Extract product title and price from HTML using BeautifulSoup's 'find' method. Clean and format the data.

4. Creating a CSV File

Create a CSV file to store extracted data using the 'csv' library. Write header and initial data.

5. Continuously updating the CSV

Append new data (title, price, and date) to the existing CSV file with 'a+' mode in the 'open' function.

6. Checking Price and Sending Email Notifications

Repeatedly check T-shirt price using 'check_price' function. Trigger email notification if price falls below a threshold.

7. Email Notifications

Send email notifications using the 'send_mail' function. Connect to Gmail via SMTP and send predefined subject and message.

8. Continuous Price Monitoring

Continuously monitor the Amazon page with an 80 second delay between each check. Send email notification when price drops below the threshold.

9. Data Analysis (Optional)

Analyze extracted data further using pandas. Sample code snippet for reading CSV file into a DataFrame is provided.

Conclusion

This project demonstrates how analysts can automate price tracking on Amazon using Python. By leveraging web scraping and basic data manipulation, analysts can stay informed about price changes and make informed purchasing decisions.
